## 1. Executive Summary

**ORVACT Tier 3** is a proposed interstellar scientific probe capable of reaching Proxima Centauri (4.24 light-years) within 40–50 years. This is **not a near-term product**. It is a **long-term technology horizon** that becomes viable ONLY after successful validation of Tiers 0, 1, and 2.

**Investment Thesis:** Corporations investing in ORVACT are not funding a single product. They are funding a **validated pathway to interstellar capability**. Tier 3 is the ultimate destination, but value is created at every tier along the way.

**Critical Dependency:** Tier 3 propulsion (gravitomagnetic thrust) **must be experimentally confirmed** in Tier 0–2. If the effect is not validated, Tier 3 defaults to conventional ion propulsion with significantly reduced performance.

**This document defines the horizon, not the commitment.** Investors should understand: Tier 3 happens ONLY if physics validates it.

## 3. Tier 3 Specification (Conditional)

### 3.1. Mission Profile
| Parameter | Target | Contingency (if gravitomagnetic unconfirmed) |
|-----------|--------|---------------------------------------------|
| Destination | Proxima Centauri (4.24 ly) | Heliopause only (100–150 AU) |
| Cruise velocity | 0.1c (30,000 km/s) | 0.001c (300 km/s) |
| Time to target | 40–50 years | 1000+ years (not viable) |
| Spacecraft mass | 80–100 t (dry) | 5–10 t (reduced payload) |
| Payload | 50 kg scientific instruments | 10 kg (miniaturized) |
| Communication | Laser 1 kW + 5 m antenna | Radio X-band (lower rate) |
| Data rate at target | 1–10 bits/sec | 0.01–0.1 bits/sec |
| Mission duration | 50 years cruise + 10 years data | 20 years (local interstellar medium) |

### 3.2. Propulsion Architecture (Two Scenarios)

**Scenario A: Gravitomagnetic Confirmed (Tier 2 Validation Success)**
| Parameter | Value |
|-----------|-------|
| Thrust mechanism | Gravitomagnetic coupling (propellantless) |
| Thrust level | 20–50 kN continuous |
| Acceleration | 0.001–0.01 m/s² |
| ΔV capability | Unlimited (bound by power, not propellant) |
| Power source | Nuclear fission 10–50 MWe |
| Heat rejection | Droplet radiator 500–1000 m² |
| Feasibility | **Viable for interstellar** |

**Scenario B: Gravitomagnetic NOT Confirmed (Tier 2 Failure)**
| Parameter | Value |
|-----------|-------|
| Thrust mechanism | Ion thrusters (conventional) |
| Thrust level | 0.5–5 N continuous |
| Acceleration | 0.00001 m/s² |
| ΔV capability | ~50 km/s (with propellant) |
| Power source | Nuclear fission 1–10 MWe |
| Heat rejection | Panel radiator 100–200 m² |
| Feasibility | **NOT viable for interstellar** (heliopause max) |

**Investor Clarity:** Tier 3 as described (Proxima Centauri) is **ONLY possible in Scenario A**. Scenario B reduces the mission to a high-end Voyager-class probe.

## 5. Technology Risks (Transparent Assessment)

| Risk | Probability | Impact | Mitigation |
|------|------------|--------|------------|
| Gravitomagnetic effect does not exist | 40–60% (unknown physics) | Tier 3 impossible | Early validation in Tier 0–2; pivot to ion propulsion |
| Gravitomagnetic effect exists but doesn't scale | 20–30% | Tier 3 reduced capability | Conservative thrust estimates in Tier 2 design |
| Nuclear power insufficient for 0.1c | 10–20% | Cruise velocity reduced | Modular reactor scaling; fusion backup (long-term) |
| Interstellar dust erosion at 0.1c | 30–40% | Spacecraft damage | Whipple shielding + magnetic deflection + redundant systems |
| Communication failure at 4.24 ly | 20–30% | Data loss | Redundant transmitters + error correction + multiple ground stations |
| Mission timeline exceeds funding horizon | 50–60% | Program cancellation | International consortium (multi-generation commitment) |

**Investor Note:** The single largest risk is **physics uncertainty**. This is why Tier 0–2 validation is non-negotiable. Do not fund Tier 3 before Tier 2 succeeds.
