# ORVACT Project

Orbital Rotational Vector Axial Counter-rotating Torque

## Status
Experimental. Tier 0 (Proof of Concept) in development.

## Abstract
ORVACT is an open-source platform for gyroscopic propulsion research. The system utilizes counter-rotating fluid-filled rotors with Tesla valve channels and magnetohydrodynamic (MHD) interaction to generate vector thrust and stabilize orientation via precession control.

## Key Features
- Counter-rotating fluid rotors (mercury/conductive fluid)
- Tesla valve passive flow control channels
- Axial electromagnetic stator with vector field control
- Static central shaft with integrated cooling
- Modular containment design

## Architecture
- Configuration: Dual counter-rotating disks, axial stator, static central shaft
- Working Fluid: Mercury (industrial), Gallium alloys or conductive simulants (prototype)
- Channel Design: Tesla valve geometry for passive flow rectification
- Control: Vector thrust via magnetic field modulation and gyroscopic precession
- Cooling: Passive convection (atmosphere) / Active radiator (vacuum)

## Development Tiers
- Tier 0: Garage prototype (2m diameter, proof of physics)
- Tier 1: Atmospheric vehicle (10m diameter, commercial)
- Tier 2: Lunar missions (8-10m diameter, piloted)
- Tier 3: Interstellar probe (4-6m diameter, autonomous)
